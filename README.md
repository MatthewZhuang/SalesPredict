# SalesPredict
Sales Prediction

本项目来自于真实的商业需求，根据国外某电子产品厂家的历史销售数据，库存数据，促销数据，应用统计学习方法对该厂的业务提出改进。

数据：
  1. 按月库存数据（各产品及总量）
  2. 按月销售数据（各产品分地区销量及总量）
  3. 促销数据（参与促销产品及相关促销结果）
  
最终实现功能：
  1. 对未来销售做出预测（时序数据分析）
  2. 提出较优绑定销售产品（关联数据挖掘）
  3. 库存调优（基于销售预测的库存管理建议）
